## Tech Stack

ROS2 | Python | Computer Vision | Robot

## Overview

This project implements an autonomous robotic system using the Mini Pupper platform, combining line following, real-time color recognition, and location-triggered behaviors.

## Key Contributions

- Implemented real-time line following and color-based decision making using onboard vision processing.
- Designed a behavior-trigger system enabling dynamic actions (for example, dance routines and directional movement) based on visual input.
- Integrated IMU and odometry for motion control, and reduced heading drift through PID tuning.
- Built scalable ROS2 packages and a launch system for multi-node coordination and long-term extensibility.
